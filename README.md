# React Redux Starter
A simple starter to get you going with developing React applications with Redux!

**Demo:** https://www.gigacore.in/demos/react-redux-demo/

### Under the hood

* React
* Redux
* SASS

### Get it runnin'!
* git clone
* cd React-Redux-Starter
* npm i
* npm start
